# Isotope Search Toolset Injection

Filter a masonry view by search bar input.

Built on [toolset-injection-boilerplate](https://github.com/warrenwhipple/toolset-injection-boilerplate).

_Deploying requires at least an [entry level purchase](https://toolset.com/buy/) of the Toolset plugin for WordPress._

## Dependencies

- [Isotope](https://isotope.metafizzy.co/) layout
- [Fuse](https://fusejs.io/) fuzzy search
- [StickyBits](https://dollarshaveclub.github.io/stickybits/) sticky search bar

## Import

Navigate to your `WordPress Dashboard` > `Toolset Export/Import` > `Views` > `Import Views`.

Upload `dist/toolset-import.xml` and import.
